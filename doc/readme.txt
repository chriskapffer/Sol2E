This folder contains a sandcastle help file builder project file, which if executed generates the documentation for the Sol2E game engine. The generated output goes into generated.

The most important documentation formats are:



Sol2E_Documentation.chm

A Microsoft compiled html help file. Best way to read the docs.



Index.html (fond in generated folder)

Starting point for documentation as web pages.



Install_Sol2E_Documentation.bat (fond in generated folder)

A script that installs the documentation into the systems Help Library Manager. This makes it accesssible from within Visual Studio.
