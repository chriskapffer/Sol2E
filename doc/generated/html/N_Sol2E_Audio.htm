Sol2E.Audio Namespace
Documentation of Sol2E Game Engine generated by Sandcastle


The audio subsystem, which provides audio related components to use in a game, as well as the AudioSystem class, which handles the logic of all related components.
Classes
  Class Description
AudioListener
A Component, which marks a listener entity but doesn't contain any fields.
AudioSource
A Component, which contains following fields: AssetName, Is3D, IsLooped, IsPlaying, IsPaused, IsStopped, Volume, Pitch and Pan.
AudioSystem
The audio system. For general explanatino on what a system does, see the documentation of IDomainSystem. Components used by this system are Transform, Movement, and AudioSource and AudioListener. The system updates the positions of AudioListener and AudioSources and set the sources play state.
