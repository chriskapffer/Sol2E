The input subsystem, which provides the InputSystem class, which updates the input states of mouse and keyboard devices. It also handles the invocation of input dependet scripts.
Classes
  Class Description
ComplexInputCondition
A complex input condition, which consists of a collection of simple input conditions. All subconditions have to be met in order for IsMet to return true.
InputDevice
Static wrapper class around xna's keyboard and mouse. Keeps track of previous and current state, thus can react to state changes.
InputScript
Abstract script, which gets invoked if one or more input conditions are met.
InputSystem
The input system. For general explanatino on what a system does, see the documentation of IDomainSystem. Components handled by this system are ScriptCollection<InputScript>. The system updates the input device states and invokes input script actions, if their input condition is met.
SimpleInputCondition
A simple input condition, which is defined by an input source and an input state. If the specified source is in the specified state then the condition is met.
